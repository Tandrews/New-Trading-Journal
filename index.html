📊 How Portfolio Tracking Works:

  • 🏦 Starting Capital: Your initial account balance
  • 💰 Current Balance: Updates automatically with trade P&L
  • 📈 Total Return: Current Balance - Starting Capital
  • 📊 Return %: (Total Return / Starting Capital) × 100
  • 📉 Max Drawdown: Largest peak-to-valley decline
  • 📏 Sharpe Ratio: Risk-adjusted return metric
